What's PillarsJS ?

PillarsJS es un Framework para hacer Web's en Node.JS.

With Pillars you can configure a HTTP Server in a few steps, build a simple Website, or a complete and complex Application with Model View Controller.

We invite you to read the PillarsJS Scopes, and then you start with the earliest examples.

We hope you enjoy it!!

Debes tener Node.JS y MongoDB instalados previamente. Puedes encontrar instrucciones de instalación aquí.

Hello World

The first step is declare all libraries; formwork, Pillar and Beam. After we create the HTTP Server, the Pillar and the Beam. 


	// Require of necesary libraries
	var formwork = require('./lib/formwork'); 
	var Pillar = require('./lib/Pillar');
	var Beam = require('./lib/Beam');

	//Creating HTTP Server on port 82
	var server = formwork(82); 

	//Routing 
	var miPillar = new Pillar({
		id : "main",
		path: "/"
	});

	//
	var myBeam = new Beam("mainBeam",{}, function (){
		var gw = this;	
		gw.send("Hello World");
	});

	myPillar.addBeam(miBeam);
	server.addPillar(myPillar);

An expert developer writes the code above as follows:


	var formwork = require('./lib/formwork'); 
	var Pillar = require('./lib/Pillar');
	var Beam = require('./lib/Beam');

	var server = formwork(82); 

	server.addPillar(new Pillar({id: 'main'})
		.addBeam(new Beam("mainBeam",{},function(){
			var gw = this;	
			gw.send("Hello World");
		}))
	);

Website One Page HTML

In this example, we are goingo to declarate the static file server, so we need it to server a html page.


	// Require of necesary libraries
	var formwork = require('./lib/formwork'); 
	var Pillar = require('./lib/Pillar');
	var Beam = require('./lib/Beam');

	//Creating HTTP Server on port 82
	var server = formwork(82); 

	server.addPillar(new Pillar({id:'staticfiles'})
		.addBeam(new Beam('uploads',{path:'/uploads/*:path',directory:'./uploads'},beams.directory))
	);
